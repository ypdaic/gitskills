启动ZK服务:        ./zkServer.sh start
查看ZK服务状态:     ./zkServer.sh status 
停止ZK服务:        ./zkServer.sh stop 
重启ZK服务:        ./zkServer.sh restart

客户端连接服务器命令:
zkCli.sh -server 127.0.0.1:2181

显示根目录下、文件： ls / 使用 ls 命令来查看当前 ZooKeeper 中所包含的内容
显示根目录下、文件： ls2 / 查看当前节点数据并能看到更新次数等数据
创建文件，并设置初始内容： create /zk "test" 创建一个新的 znode节点“ zk ”以及与它关联的字符串  [-e] [-s] 【-e 零时节点】 【-s 顺序节点】
获取文件内容： get /zk 确认 znode 是否包含我们所创建的字符串  [watch] (添加watch监听)
修改文件内容： set /zk "zkbak" 对 zk 所关联的字符串进行设置
删除文件： delete /zk 将刚才创建的 znode 删除，如果存在子节点删除失败 
递归删除：rmr  /zk将刚才创建的 znode 删除，子节点同时删除
退出客户端： quit 
帮助命令： help

 配置文件

 storage:
    journal:enabled: true
    dbPath: /data/zhou/mongo1/
    ##是否一个库一个文件夹
    directoryPerDB: true
    ##数据引擎
    engine: wiredTiger
    ##WT 引擎配置
    wiredTiger:
        engineConfig:
        ##WT 最大使用cache（根据服务器实际情况调节）
        cacheSizeGB: 1
        ##是否将索引也按数据库名单独存储
        directoryForIndexes: true
        journalCompressor:none （默认 snappy）
        ##表压缩配置
        collectionConfig:
            blockCompressor: zlib (默认 snappy,还可选 none、zlib)
        ##索引配置
        indexConfig: prefixCompression: true


 压缩算法 Tips:

     性能: none > snappy >zlib
     压缩比:zlib > snappy > none

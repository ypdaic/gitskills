定义 a=10,b=20

+ 加法 `expr $a + $b`
- 减法 `expr $a - $b`
* 乘法 `expr $a \* $b`
/ 除法 `expr $a / $b`
% 取余 `expr $b % $a`
= 赋值  a=$b 将变量b赋值给a
== 相等。用于比较两个数字，相同则返回true
!= 不相等，用于比较两个数字，不相同则返回 true
